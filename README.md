# 📊Food-Delivery-SQL-Analytics-Project
---
 
 ## 📌 Project Overview

This project focuses on analyzing a food delivery dataset using SQL to extract meaningful business insights.
The goal is to understand customer behavior, order patterns, and growth metrics through real-world analytical queries.

---

## 🎯 Objectives

- Identify top-performing restaurants by cuisine
- Analyze customer acquisition trends over time
- Track user behavior and order frequency
- Measure organic vs promoted customer growth
- Support data-driven decision making

---

## 🛠️ Tools & Technologies

- SQL (MySQL Workbench)
- SQL Joins
- Aggregations
- Subqueries
- Window Functions
- Common Table Expressions (CTEs)

---

## 📂 Dataset 

The dataset contains food delivery order details including:

- "customer_code"
- "restaurant_id"
- "cuisine"
- "placed_at"
- "promo_code_name"
- "order_status"

---

## 📈 Key Insights

- Customer acquisition trends vary significantly across dates
- A portion of users rely entirely on promo-based purchases
- Organic customers contribute significantly to long-term growth
- Identifying repeat behavior helps in targeted engagement strategies

---

## 🚀 Learnings

- Strong understanding of SQL window functions
- Hands-on experience with CTEs and advanced filtering
- Ability to translate business problems into SQL queries

---

## 🙌 Conclusion

This project demonstrates how SQL can be used to solve real-world business problems and generate actionable insights in a food delivery context.
